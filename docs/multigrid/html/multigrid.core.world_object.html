class multigrid.core.world_object.WorldObj(type: str = 'empty', color: str | None = None)[source]#

Bases: ndarray

Base class for grid world objects.

The state is a 4-dimensional integer array, indexed as follows: * 0: object type * 1: object color * 2: object state (i.e. open/closed/locked) * 3: mixed-radix integer encoding of contained object

Attributes:
typestr

Return the name of the object type.

colorstr

Return the name of the object color.

statestr

Return the name of the object state.

containsWorldObj or None

Return the object contained by this object.
