class multigrid.utils.misc.PropertyAlias[source]#

Bases: property

A class property that is an alias for an attribute property.

Instead of:

@property
def x(self):
    self.attr.x

@x.setter
def x(self, value):
    self.attr.x = value

we can simply just declare:

x = PropertyAlias('attr', AttributeClass.x)
